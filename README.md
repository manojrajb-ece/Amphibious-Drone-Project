# Amphibious Drone Project

## Overview
An Arduino-based Amphibious Drone designed to operate in both air and water environments. The project focuses on developing a dual-mode drone system capable of navigation, monitoring, and control across different environments.

## Features
- Arduino-based flight controller  
- Dual air and water operation  
- Brushless motors and ESC integration  
- Wireless transmitter & receiver control  
- Motor calibration and setup configuration  

## Components Used
- Arduino Uno  
- Brushless Motor  
- Electronic Speed Controller (ESC)  
- LiPo Battery  
- Propeller  
- Drone Frame  
- Transmitter & Receiver  

## Software Used
- Arduino IDE  

## Applications
- Flood rescue operations  
- Environmental monitoring  
- Coastal surveillance  
- Marine exploration  
- Search and rescue missions

Flight controller implementation inspired by online educational resources and YouTube tutorials used for learning and educational purposes.

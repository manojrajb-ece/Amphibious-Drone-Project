# Amphibious Drone Project

## Overview
An Arduino-based Amphibious Drone designed to operate in both air and water environments.

## Team Members
- Lohith M
- Manoj Raj B
- Pavan Kumar S
- L Suhas

## Features
- Arduino based flight controller
- Dual air and water operation
- Brushless motors and ESC
- Wireless transmitter & receiver

## Components Used
- Arduino Uno
- Brushless Motor
- ESC
- LiPo Battery
- Propeller
- Drone Frame

## Software Used
- Arduino IDE

## Applications
- Flood rescue operations
- Environmental monitoring
- Coastal surveillance
- Marine exploration
